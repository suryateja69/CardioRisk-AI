# ❤️ CardioRisk AI

CardioRisk AI is a machine learning-based heart disease risk assessment web application built using Python, Scikit-learn, and Streamlit.

## 🚀 Features
- Predicts heart disease risk from patient health indicators
- Interactive Streamlit web interface
- Risk percentage score
- Low / Moderate / High risk classification
- Patient summary and health recommendation
- Educational medical disclaimer

## 🧠 Machine Learning Model
- Algorithm: Logistic Regression
- Scaling: StandardScaler
- Dataset: Heart Disease Dataset
- Accuracy: Around 80–87%

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## 📂 Project Files
- `app.py` - Streamlit web app
- `cardiorisk_model.pkl` - Trained ML model
- `scaler.pkl` - Feature scaler
- `requirements.txt` - Required libraries
- `README.md` - Project documentation
